# KefirControl

Este directorio agrupa las pantallas del modulo KefirControl por modulo funcional.

- `_shared`: layout, navegacion y piezas compartidas.
- `inicio`: dashboard y entrada principal.
- `productos`: catalogo de productos elaborados.
- `lotes`: gestion de lotes.
- `ordenes`: ordenes de produccion.
- `auditoria`: kardex, trazabilidad y auditoria.
- `nodulos`: control de nodulos.
- `reportes`: analisis y reportes.
- `inventario-produccion`: stock e inventario de planta.
- `calidad`: evaluacion de calidad.
- `costos`: rentabilidad y costos.
- `rendimientos`: rendimientos de produccion.

La estructura conserva las rutas visibles actuales; mover codigo aqui no debe cambiar la logica ni el UX/UI.
